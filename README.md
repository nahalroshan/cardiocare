# Cardio Care - Heart Disease Management App

## Description

Cardio Care is a comprehensive heart disease management app that integrates disease prediction, community forums, dietary plans, and exercise recommendations.

## Features

### Disease Prediction

Utilizes machine learning to predict heart disease risk based on user input.

### Firebase Authentication

Ensures secure user authentication for access to personalized features.

### Community Forum

Enables users to share experiences, concerns, and doubts related to heart health.

### Dietary Plans

Provides personalized dietary plans tailored to individual health needs.

### Exercise Recommendations

Offers exercise routines and recommendations for maintaining heart health.

## Technologies Used

- **Frontend**: React.js, Firebase Authentication
- **Backend**: Flask (Python), Machine Learning for disease prediction
- **Database**: Firebase Firestore for storing user data
- **Deployment**: Frontend hosted on Firebase Hosting, Backend hosted on Flask server

## Getting Started

To run the application locally:

1. Clone the repository for both frontend and backend.
2. Set up Firebase authentication for the frontend and Firestore for the backend.
3. Install dependencies for both frontend and backend.
4. Run the frontend and backend servers.
5. Access the application in your browser.
